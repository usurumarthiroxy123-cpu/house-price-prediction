# House Price Prediction

This project predicts house prices based on area using Python and Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Description
A simple Linear Regression model is trained on sample house data to predict prices based on house area.

## How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn
2. Run the program:
   python house_price_prediction.py
